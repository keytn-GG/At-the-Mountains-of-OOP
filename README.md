## このアプリについて
このアプリは、Javaでオブジェクト指向プログラミング（OOP）を学ぶために作成されました。  
テーマは『狂気の山脈にて』（"At The Mountains Of Madness" 著 H.P.ラヴクラフト）のクトゥルフ神話TRPG（CoC）風ゲームブックで、

- キャラクタークリエイト
- ダイスによる技能判定
- 体力などのパラメータ管理
- シーン管理
- エンディング管理

などを設計・実装予定です。

学習目標は、  

1. OOPにおける「オブジェクトの役割」を体感する！
2. 役割を意識した設計をできるようになる！
3. 知らないJava APIや組み込みメソッドを試す！

です。  

## 遊び方
Eclipseを使用している前提の遊び方です。

### プロジェクトをインポートする

1. Githubリポジトリページから、zipファイルをダウンロードします。
2. zipファイルを解凍して、それをEclipseワークスペースに指定しているディレクトリに移動させます。
3. Eclipseのパッケージエクスプローラー上で右クリック。
4. 「インポート」
5. 「一般」
6. 「既存プロジェクトをワークスペースへ」
7. 「ルートディレクトリの選択」で、解凍したプロジェクトのルートディレクトリを選択。  
( おそらくAt-the-Mountains-of-OOP-main )
8. 完了

### 遊ぶ
mainパッケージの、Main.javaを実行してください。
